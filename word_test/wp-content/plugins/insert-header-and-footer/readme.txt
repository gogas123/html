Insert custom codes in header and footer.

== Description ==

Insert Header and Footer plugin allows to insert custom codes inside head tag and above body tag. it supports custom HTML, JavaScript and CSS code. this plugin uses latest WordPress code editor so you can past or type bugs free custom code. No need to edit theme files for custom code.

= Steps to Setup =

<ol>
	<li>Install and active Insert Header and Footer plugin.</li>
	<li>Open page Header and Footer</li>
	<li>Type or past head code in first editor and footer code in second editor.</li>
	<li>Save settings.</li>
	<li>Done !!</li>
</ol>

Insert Header and Footer is a lightweight plugin so it will not affect performance of your website. it is easy to use plugin so you can setup within 30 seconds.

Insert Header and Footer plugin print codes as it is so you should use Script tag for JavaScript code and Style tag for CSS code.

== Installation ==

<ol>
	<li>Download plugin zip file.</li>
	<li>Open Dashboard of your WordPress website.</li>
	<li>Open: Plugins > Add New > Upload Plugin > Browse... (select downloaded zip file) > Install Now</li>
	<li>Now simply activate plugin.</li>
</ol>

== Frequently Asked Questions ==

= Where can i add custom code =
Please open 'Dashboard' and then 'Header and Footer' menu. here you will see two editors. first editor is for head code and second editor it for footer code. simply type or past your codes and save settings. you should use Script tag for JavaScript code and Style tag for CSS code because this plugin will print code as it is.
